# TokyoR98-BeginneR

2022年4月16日に行われた第98回R勉強会@東京（#TokyoR）の初心者セッションで用いたコードです。
https://tokyor.connpass.com/event/244200/

tidymodelsを用いた機械学習モデルの構築と、DALEXを用いた機械学習モデルの解釈を行っています。
